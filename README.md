## ✨ Visão Geral

O projeto consiste em uma página web responsiva que simula o ambiente digital da biblioteca da faculdade UniFECAF. O objetivo é oferecer uma experiência intuitiva para os alunos, permitindo conhecer os serviços, navegar pelo catálogo de livros e acessar as unidades de aprendizado em desenvolvimento front-end.

---

## 🚀 Tecnologias Utilizadas

| Tecnologia | Descrição |
|------------|-----------|
| **HTML5** | Estrutura semântica da página |
| **CSS3** | Estilização visual, layout responsivo e animações |
| **Flexbox/Grid** | Organização dos cards e seções |
| **Media Queries** | Adaptação para dispositivos móveis |

---

## 📖 Funcionalidades

- ✅ Apresentação institucional da biblioteca virtual
- ✅ Galeria de imagens com destaques do acervo
- ✅ Catálogo de livros exibido em cards interativos
- ✅ Seção com 3 unidades de estudo (HTML/CSS, Responsividade, JavaScript)
- ✅ Área de contatos com informações institucionais
- ✅ Menu de navegação âncora (scroll suave para as seções)
- ✅ Design responsivo (mobile first)
